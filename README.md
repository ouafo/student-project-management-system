# student-project-management-system
A web-based project management system designed for students to manage group projects, assign tasks, set priorities and collaborate.
## Setup Project
1. Clone the repository: git clone https://github.com/USERNAME/student-project-management-system.git
2. Go to the project folder
3. Create your branch with command git branch your_branch_name 
4. go to your branch with command git checkout your_branch_name
5. Work on your feature
6. Commit your changes with followings commands:
      git add .
     git commit -m "your message"
7. Push your branch with command git push -u origin your_branch_name
